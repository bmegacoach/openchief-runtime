## Technology Architecture

### AI Stack
- **Primary LLM:** Qwen 3.6 235B via OpenRouter (open-weight, auditable)
- **Agent OS:** OpenFang — Rust-based Agent Operating System (14 crates, v0.5.5)
- **RAG Pipeline:** Semantic memory with vector embeddings, cosine similarity search
- **Embedding:** Ollama nomic-embed-text (768 dimensions, local)
- **Knowledge Graph:** Entity-relation store with confidence scoring
- **Memory Backend:** SQLite (local) or PostgreSQL + pgvector (production)
- **Model Catalog:** 197 models available, 28 from configured providers

### Infrastructure
- **Frontend:** React + TypeScript + Vite + Tailwind CSS
- **Backend:** Supabase (PostgreSQL, Auth, Edge Functions)
- **Payments:** Stripe via Supabase Edge Functions
- **Agent API:** OpenFang REST API (port 4200) with A2A protocol
- **Fleet:** Desktop + Mini PC (192.168.100.90), SSH port 2222
- **Observability:** OpenFang dashboard (Alpine.js SPA)
- **Voice Gateway:** faster-whisper transcription → Groq fallback → Claude/Phi intent classification

### Blockchain Stack
- **Primary Chains:** Base, Solana, Ethereum
- **Cross-chain:** LayerZero protocol (24 networks)
- **Smart Contracts:** ERC-20 (USDca), ERC-721/1155 (RWA NFTs), DAO governance

### Security (3 Layers)
- Layer 1: Channel ACL + heartbeat monitoring
- Layer 2: Prompt injection scanner (HIGH/MEDIUM/LOW)
- Layer 3: Outbound secret redaction (API keys, tokens stripped)
